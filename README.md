# Simple XML validator FX

## О проекте

В составе проекта входят:
* Утилита для выгрузки XSD-схемы и Schematron-файлов с git.minzdrav.gov.ru
* Утилита для перевода Base64-строки в XML-файлы
* Утилита для проверки XML-файла на корректность по XSD-схемам и Schematron-файлам, выгруженным с git.minzdrav.gov.ru

## Сборка проекта

Для сборки потребуется:
* Java 21
* Maven

Команда сборки (из корня проекта):

```
mvn clean install
```

В результате в папке `target` появится файл `simple-xml-validator-fx-X.X.X.jar`

## Запуск проекта

Для запуска проекта требуется Java 21 с установленными модулями JavaFX.
Запуск выполняется командой:

```
java -jar simple-xml-validator-X.X.X.jar
```

Или скриптом `run-validator.sh`

## Используемые сторонние библиотеки

* **XSD** – валидация выполняется встроенными библиотеками javax.xml.*
* **Schematron** – валидация выполняется сторонней библиотекой [ph-schematron](https://phax.github.io/ph-schematron/)

## Дополнительная информация (для разработчиков)

Расширить логирование валидатора можно параметром ```-Djaxp.debug=true```, указав его в VM options.
